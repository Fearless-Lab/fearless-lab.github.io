# Data Visualization Practice

GSAP과 Recharts를 연습하기 위한 React 프로젝트

## 주요 내용

- GSAP를 이용한 기본 애니메이션
- Recharts로 라인 차트 구현
- Tailwind CSS로 스타일링
- React Router로 라우팅 처리

## 기술 스택

React, Vite, TypeScript, GSAP, Recharts, Tailwind CSS, React Router
